# 데이터 종류
- String
- Number
- Boolean
- Undefined
- Null
- Object
- Array

## String
- 따옴표, 큰따옴표, 백틱.
```javascript
let myName = "HEROPY";
let email = 'thesecon@gmail.com';
let hello = `Hello ${myName}`;
```

## undefined
- 값이 할당되지 않은 상태

```javascript
let undef;
let obj = {abc: 123};
```

## Null
- 값이 의도적으로 비어있음을 의미.


```javascript
let empty = null;

```

## Object
- 여러 데이터를 키 밸류 형태로 저장


```javascript
let user = {
  name: 'HEROPY',
  age: 85,
  isValid: true
};

```


## Array
- 여러 데이터를 순차적으로 저장


```javascript
let fruits = ['Apple', 'Banana', 'Cherry'];

```
