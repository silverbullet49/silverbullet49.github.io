# 모서리둥글게(border-radius)
- 요소의 모서리를 둥글게 깎음.
- 기본값: 0(깎는거없음)
- 단위: px, em, vw 등 단위로 지정
```css
.container {
  /* left-top, right-top, right-bottom, left-bottom 순 */
  border-radius: 0 10px 0 0;
}
```
