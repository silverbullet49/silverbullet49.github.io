# 크기계산(box-sizing)
- 요소의 크기 계산 기준을 지정
- 기본값: content-box(요소의 내용(content)으로 크기 계산)
- border-box: 요소의 내용 + padding + border로 크기 계산

## border-box
- 요소에 border, padding을 추가하여 크기가 커지는 것을 막고 싶을 때 사용
- 요소의 내용(content) + padding + border의 총합이 width, height에 지정한 값이 되도록 맞춘다.(줄어듦)
