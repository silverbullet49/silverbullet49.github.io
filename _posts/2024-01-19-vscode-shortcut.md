# 단축키

- ctrl + B = 사이드바 열기/닫기
- ctrl + p = 파일 열기
- ctrl + shift + p = 모든 명령을 표시(검색)
- ctrl + w = 탭 닫기
- ctrl + f = 텍스트 찾기
- ctrl + alt + f = 찾은 텍스트 바꾸기

##
- alt + up = 해당 줄이 위로 이동
- alt + down = 해당 줄이 아래로 이동
- alt + shift + down = 해당 줄을 아래에 복붙
- alt + shift + up = 해당 줄을 위에 복붙

##
- shift + tab = 내어쓰기

##
- ctrl + / = 주석

##
- ctrl + \ = 창 분할하여 탭 복제
